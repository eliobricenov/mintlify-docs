## Package Installation

Install the Eco React SDK using your preferred package manager:

<Tabs>
  <Tab title="npm">
    ```bash
    npm install @eco-foundation/react-sdk
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn add @eco-foundation/react-sdk
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add @eco-foundation/react-sdk
    ```
  </Tab>
</Tabs>

## Peer Dependencies

The Eco React SDK requires the following peer dependencies. Make sure to install them in your project:

<Tabs>
  <Tab title="npm">
    ```bash
    npm install @tanstack/react-query wagmi @wagmi/core viem @eco-foundation/routes-sdk @eco-foundation/routes-ts
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn add @tanstack/react-query wagmi @wagmi/core viem @eco-foundation/routes-sdk @eco-foundation/routes-ts
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add @tanstack/react-query wagmi @wagmi/core viem @eco-foundation/routes-sdk @eco-foundation/routes-ts
    ```
  </Tab>
</Tabs>


After installation, proceed to the [Setup](/essentials/setup) guide to configure the SDK in your application.
